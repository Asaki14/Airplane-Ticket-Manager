## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | 图标与短文本微间距 |
| sm | 8px | 紧凑控件间距、标签组间距 |
| md | 16px | 卡片内默认内边距、模块默认 gap |
| lg | 24px | 区块内边距、主布局基础分隔 |
| xl | 32px | 大区块间距、Hero 与主内容过渡 |
| 2xl | 48px | 页面级分区断点间隔 |
| 3xl | 64px | 大屏页面主节奏留白 |

Exceptions: 交互触达目标最小高度固定 44px（输入框、按钮、导航项），不随内容缩小（source: `src/styles/globals.css`）

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 | 1.5 |
| Label | 14px | 600 | 1.4 |
| Heading | 20px | 600 | 1.2 |
| Display | 28px | 600 | 1.2 |

说明：本 phase 仅允许 4 个字号（14/16/20/28）与 2 个字重（400/600），避免视觉层级继续膨胀；与现有 tokens 对齐（source: `src/styles/tokens.css`，line-height 为本规范新增约束）。

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #F7F4ED | 页面背景、主阅读底色、移动端主滚动背景 |
| Secondary (30%) | #E5EDE8 | Hero 渐变底、卡片底色、导航/筛选容器底 |
| Accent (10%) | #0B7A75 | 主 CTA 按钮、focus-visible 环、关键可执行链接 |
| Destructive | #C44936 | 仅用于明确风险动作（删除/归档/不可逆确认） |

Accent reserved for: `应用筛选` 按钮、`查看详情与票规` 按钮、`去来源页继续购买` 按钮、键盘 focus-visible 高亮。禁止用于普通正文、非关键标签和大面积背景。
